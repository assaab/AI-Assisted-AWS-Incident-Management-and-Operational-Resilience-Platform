# RACI

| Activity | Operator | Approver | Checkout Owner | SRE Lead | Executive |
| --- | --- | --- | --- | --- | --- |
| Acknowledge incident | R | I | C | A | I |
| Validate evidence | R | C | C | A | I |
| Approve rollback | C | A/R | C | C | I |
| Execute rollback | R | I | C | A | I |
| Verify recovery | R | I | C | A | I |
| Publish executive summary | C | I | C | A | R |

R = responsible, A = accountable, C = consulted, I = informed.
